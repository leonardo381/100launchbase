# Flowbite Pro Blocks Catalog

## Purpose

Private index for local Flowbite Pro reference paths and block counts.
Do not paste proprietary Flowbite Pro source code into this document.

## Core Rules

- Raw Flowbite Pro references live only under `.flowbite-pro-reference/raw/`.
- `.flowbite-pro-reference/` must remain gitignored.
- Raw paths follow original Flowbite Pro category names (slug form).
- LaunchBase usage targeting is assigned later during block selection, not encoded by folder naming hacks.
- `_unmapped/` is reserved for any real pasted content that cannot be confidently classified.

## Selection Notes For LaunchBase

- Auth surfaces may use `marketing-ui/login-forms`, `register-forms`, `reset-password-forms`, and `account-recovery`.
- Dashboard placeholders should be assembled from available `application-ui` categories (shells, navbars, tables, CRUD layouts), not invented buckets.
- Ecommerce payment and checkout block references are visual-only at this phase. Billing implementation should continue to use Stripe Checkout and Stripe Customer Portal server-side unless explicitly changed later.
